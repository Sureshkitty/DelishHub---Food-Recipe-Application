DelishHub - Food Recipe Application
Description:
DelishHub is a dynamic web application designed for food enthusiasts to discover, create, and share recipes. Key features include user authentication for secure sign-in and sign-up, an extensive recipe database powered by an external API, and a responsive design that ensures seamless use on both desktop and mobile devices. Users can easily browse, search, and contribute recipes, enhancing their culinary experience.

Technologies Used:
HTML, CSS, JavaScript, PHP, API
